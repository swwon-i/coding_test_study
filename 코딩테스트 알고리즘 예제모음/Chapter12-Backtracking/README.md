# 12. 백트래킹
## 12-1. 백트래킹과 백트래킹 알고리즘 개념
완전 탐색은 모든 경우의 수를 탐색하는 방법이므로 대부분의 경우 비효율적
### 백트래킹이란?
어떤 가능성이 없는 곳을 알아보고 되돌아가는 것을 백트래킹
### 백트래킹 알고리즘이란?
해가 될 가능성이 없는 곳에서는 되돌아가고, 가능성이 있는 곳을 탐색하는 알고리즘
깊이우선탐색도 백트래킹을 활용(해가 있을 가능성이 없는 경우)
### 유망 함수란?
답이 될 가능성을 판단하는 것
1. 유요한 답의 집합을 정의
2. 위 단계에서 정의한 집합을 그래프로 표현
3. 유망 함수를 정의
4. 백트래킹 알고리즘을 활용해서 답을 찾음
