## 09-2 이진 트리 표현하기
배열이나 포인터로 구현
### 배열로 표현하기
- 루트 노드는 배열 인덱스 1번에 저장
- 왼쪽 자식 노드의 배열 인덱스는 부모 노드의 배열 인덱스 X 2
- 오른쪽 자식 노드의 배열 인덱스는 부모 노드의 배열 인덱스 X 2 + 1
메모리만 넉넉하면 쉽고 빠르게 구현가능
### 이진 트리 순회하기
- 전위 순회
- 중위 순회
- 후위 순회
#### 전위 순히 과정 살펴보기
부모노드->왼쪽 자식->오른쪽 자식
트리 복사할 때 많이 사용
#### 중위 순회 과정 살펴보기
왼쪽 자식->부모노드->오른쪽 자식
정렬된 순서로 가져올때
#### 후위 순회 과정 살펴보기
왼쪽 자식->오른 자식->부모 노드
