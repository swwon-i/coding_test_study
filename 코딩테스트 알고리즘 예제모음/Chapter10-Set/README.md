# 10. 집합
## 10-1 집합과 상호배타적 집합의 개념
### 집합의 개념
집합은 순서와 중복이 없는 원소들을 갖는 자료구조
### 집합의 종류
#### 상호배타적 집합
공통 원소가 없다
### 상호배타적 집합의 특성을 활용하는 분야
- 이미지 분할
- 도로 네트워크 구성
- 최소 신장 트리 알고리즘 구현
- 게임 개발
- 클러스터링 작업
## 10-2 집합의 연산
### 배열을 활용한 트리로 집합 표현하기
#### 대표 원소란
루트 노드
#### 배열로 집합을 표현하는 것이란?
배열의 인덱스는 자신을, 배열값은 부모 노드를 의미
루트 노드는 배열의값과 인덱스가 같다
#### 집합 표현 완벽 정리
##### 집합을 배열로 구현
초기 노드값 == 자신의 인덱스값
없는 노드는 -1로
### 유니온-파인드 알고리즘
합치기는 유니온, 탐색을 파인드라고 함 둘을 묶어 유니온-파인드
#### 파인드 연산
find(7)->7의 루트 노드를 찾는것
부모를 찾는것이 아닌 루트를 찾는것으로 속도가 좀 걸림
#### 파인드 연산 비용 문제, 경로 압축으로 해결
트리의 높이를 줄이기
Find한 값을 저장해 주는 것
#### 합치기 연산
두 노드의 루트 노드중 하나를 선택해서 하나의 루트를 다른 루트의 루트노드로 지정
깊이기 깊어질수록 연산 비용이 커짐
랭크라는 개념을 도입
#### 랭크란
현재 노드를 기준으로 가장 깊은 노드까지의 경로 길이
랭크가 루트노드를 작은애를 자식으로
